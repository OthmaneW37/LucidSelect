Note sur les limites de l'API

Si vous rencontrez des erreurs 429 (Too Many Requests), cela peut être dû à:

  • Un dépassement de votre limite de requêtes par minute
  • Un solde insuffisant sur votre compte API
  • Un problème avec votre clé API

Veuillez vérifier votre utilisation et vos informations de facturation sur le site du fournisseur d'API correspondant.
