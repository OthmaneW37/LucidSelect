LucidSelect

Sélectionnez du texte sur n'importe quelle page web et posez une question à l'IA.

  1. Sélectionnez du texte sur une page web
  2. Cliquez-droit et choisissez "Question à l'IA" ou utilisez le raccourci Alt+L pour répondre directement
  3. Saisissez votre question dans la popup (si nécessaire)
  4. La réponse s'affichera dans une bulle au-dessus du texte sélectionné
